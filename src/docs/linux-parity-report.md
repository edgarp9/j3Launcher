# Linux parity report

Date: 2026-06-17

Scope: Windows Win32 behavior is the reference. Linux GTK4 behavior was checked against menu commands, dialog flows, button/context-menu events, platform launch/open/copy boundaries, state persistence, and build/test gates by static code comparison, targeted tests, cross-target checks, GTK startup smoke, and an X11 GTK input smoke covering every File menu command through real menu activation or the matching Win32 accelerator plus repeated accelerator dispatch after resize/focus changes. Full side-by-side desktop manual verification remains open below.
